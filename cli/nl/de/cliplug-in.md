# Plug-ins für die Cloud Foundry-Befehlszeilenschnittstelle

*Letzte Aktualisierung: 2. Februar 2016*

Sie können die Plug-ins für die Cloud Foundry-Befehlszeilenschnittstelle (cf cli) installieren und verwenden, die im [Plug-in-Repository für Bluemix CLI](http://plugins.{DomainName}/){: new_window} verfügbar sind. Jedes Plug-in wird durch seinen Binärdateinamen, seinen vom Entwickler definierten Plug-in-Namen und durch die Befehle, die das Plug-in bereitstellt, identifiziert. Zur Installation eines Plug-ins können Sie nur den Binärdateinamen verwenden. Für jede andere Aktion können Sie den Plug-in-Namen oder einen Befehl verwenden.
{:shortdesc}
